題目： LeetCode - 20. Valid Parentheses

題目說明

給一個只含有括號的字串，求此字串是否合法。字串合法須滿足以下條件：

  • 開括號必須用相同類型的括號封閉
  • 開括號必須以正確的順序關閉

解題思路

由於閉括號必須搭配相同類型的開括號，所以可以使用 Stack，碰到開括號時直接推入，若碰到閉括號時只要看 stack.top() 是否是對應的開括號即可。
