題目： LeetCode - 1046. Last Stone Weight

題目說明

給一個陣列，每次找出最大及次大的值，若是兩者相同則消除，否則消除小的，而大的值改為兩者相減。

解題思路

  • 解法一：每次都從陣列中取出最大及次大的值，並同時將陣列該位置的值改為 0。使用 pair 儲存，first 放值，second 放 index。

  • 解法二：使用 priority queue。
