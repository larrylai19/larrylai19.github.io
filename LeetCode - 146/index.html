題目： LeetCode - 146. LRU Cache

題目說明

建立一個類似快取記憶體的 Class，及對應的一些功能。

解題思路

使用 list<pair<int, int>> 儲存資料，再利用 unordered_map<int, list<pair<int, int>>::iterator> 達到減少時間複雜度的效果。
※ 對於 put 來說，如果資料已經滿了，那新的資料就要覆蓋最久沒有調用到的資料。
